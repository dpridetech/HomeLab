## Notes
- Shebang (#!/bin/bash) tells the system which interpreter to use for the script.
- set -e makes the script exit immediately if any command fails.
- Use comments (#) and echo statements to explain steps in the script for readability.
- **Make scripts executable with:**
``` bash
chmod +x scripts/install_docker.sh
```
**Run scripts using:**
``` bash
./scripts/install_docker.sh
# or
bash scripts/install_docker.sh
# or with sudo if necessary
sudo bash scripts/install_docker.sh
```
